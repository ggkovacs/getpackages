# Get packages

## Simple example
```javascript
var gp = require('get-packages').init();
```

## Methods
* gp.init(config)
* gp.build()
* gp.get()
* gp.getCssPaths()
* gp.getAllCssPath()
* gp.getAllJsFile()
* gp.getAllDistPaths()
* gp.getBuildJs()
* gp.getImagePaths()
* gp.getFontPaths()

## Requirement
* Yii Packages command
